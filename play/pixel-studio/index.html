Hello!

Welcome to the TeCanal Pixel Studio! If you look at the top menu, you will see the level icons. Please click on Level 1 to get started.

For the best experience, please use this site on Google Chrome on a desktop computer, but this site should work on any modern web browser on any modern device.

Export Code to HTML Page

This allows you to generate a downloadable HTML file, which automatically runs your code and displays the results. You can open it up in any browser, which makes it great if you made a game and want to download it and play it whenever you want.

If you choose to not include the console in the exported page, you will not be able to see console.log() messages, or any errors. If your program does not have these, then feel free to not include console in the exported page for a more minimal look.

Share Code With URL

This allows you to create a link that contains all of your code. You can then send/share this link wherever, and if someone clicks on the link, it will open up this website with your code already loaded into the editor.
